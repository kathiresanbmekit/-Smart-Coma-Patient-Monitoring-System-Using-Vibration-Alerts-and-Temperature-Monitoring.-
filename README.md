Smart Coma Patient Monitoring System Using Vibration Alerts and Temperature Monitoring
The Smart Coma Patient Monitoring System Using Vibration Alerts and Temperature Monitoring is an advanced healthcare monitoring system designed to provide continuous observation of coma patients. Patients in a coma require constant supervision because they are unable to communicate their condition or respond to emergencies. This project aims to assist doctors, nurses, and caregivers by automatically monitoring important patient parameters and generating alerts when necessary.
The system is built using an Arduino UNO, a temperature sensor, a vibration sensor, an LCD display, and a buzzer. The temperature sensor continuously measures the patient's body temperature and sends the data to the Arduino. Maintaining normal body temperature is essential for coma patients, as sudden changes may indicate infection, fever, or other medical complications. The measured temperature values are displayed on the LCD screen, allowing caregivers to monitor the patient's condition in real time.
The vibration sensor is used to detect body movements, vibrations, or any sudden physical activity. Although coma patients generally have limited movement, detecting unexpected motion can help identify changes in their condition. The sensor can also detect accidental disturbances or movements around the patient. When abnormal movement or vibration is detected, the system immediately activates an alert mechanism.
The Arduino processes the data received from both sensors and compares the readings with predefined threshold values. If the patient's temperature exceeds the normal range or if unusual vibrations are detected, a buzzer is activated to notify medical staff or caregivers. This immediate alert system helps ensure timely medical intervention and reduces the risk of delayed treatment.
The LCD display provides a user-friendly interface by showing the current temperature readings and system status. This allows healthcare professionals to quickly assess the patient's condition without requiring complex equipment. The system operates continuously, ensuring round-the-clock monitoring and improving the reliability of patient care.
This project offers a low-cost, efficient, and easy-to-implement solution for healthcare environments such as hospitals, intensive care units (ICUs), nursing homes, and home-care settings. By integrating temperature monitoring and vibration-based activity detection into a single platform, the system enhances patient safety and supports healthcare professionals in delivering better care. The Smart Coma Patient Monitoring System demonstrates how embedded systems and sensor technology can be used to improve healthcare monitoring and provide a safer environment for critically ill patients.
